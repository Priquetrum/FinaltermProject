## How to Run Locally
To run the Tokopedia Play Clone locally, follow these steps:

#clone repository

```
  git clone https://github.com/Priquetrum/FinaltermProject
```

# Install dependencies

```
  npm install
```

#Start the server
```
  npm start
```
access at localhost:3000.
